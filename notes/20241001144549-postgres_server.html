PostgreSQL Server

Locating the database server files

Postgres Database server files are initially stored in a location referred to as data directory: 

SHOW data_directory;

  • $PGDATA is where the database files live. You can also use initdb to create the database directory: 

        initdb -D <path>

Server Management with pg_ctl

Starting and Stopping the Server

  # wait for all connections...
  pg_ctl stop -m s[mart]
  # force disconnect
  pg_ctl stop -m f[ast]
  # SIGQUIT
  pg_ctl stop -m i[mmediate]
