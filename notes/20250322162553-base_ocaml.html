Base (OCaml)

Jane Street, a company that has been using OCaml for more than 20 years, developed the code in Base for its own internal use, but from the start designed it with an eye toward being a general-purpose standard library. Like the OCaml language itself, Base is engineered with correctness, reliability, and performance in mind.

(Madhavapeddy and Minsky 2022, 3)

References:

Madhavapeddy, Anil, and Yaron Minsky. 2022. Real World Ocaml: Functional Programming for the Masses. Cambridge University Press.
